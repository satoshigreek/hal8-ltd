# HAL 8 — AI Lab

The official HAL 8 website. HAL 8 is the inference platform that turns AI agents into metered, billable infrastructure — and the lab that built the Apex Fusion blockchain.

## Deploy (GitHub Pages)

This is a single, fully self-contained `index.html` (fonts, images, and scripts are inlined — no build step).

1. Create a new repository at https://github.com/new (e.g. `hal8-site`).
2. Upload `index.html` (and this `README.md`) to the repo — drag-and-drop via **Add file → Upload files**, or push with git.
3. **Settings → Pages →** Source: **Deploy from a branch** → Branch: **main / (root)** → **Save**.
4. The site goes live at `https://<your-username>.github.io/hal8-site/`.

### Custom domain (optional)
Add a file named `CNAME` containing your domain (e.g. `hal8.io`), then set the DNS records GitHub shows under Settings → Pages.
